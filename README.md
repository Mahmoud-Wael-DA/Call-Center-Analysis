# Call Center Performance Dashboard (Excel) 📞

## Project Overview
An interactive Excel dashboard designed to monitor and analyze Call Center operations. This project transforms raw operational data into visual insights to improve response times and customer satisfaction.

## Key Metrics & Insights
- **Operational Efficiency:** Tracked total calls (1,077) and Average Call Duration (5.61 mins).
- **Response Time Analysis:** Visualized performance against SLAs (Service Level Agreements).
- **Customer Satisfaction:** Monitored Average Customer Rating (24.88) and analyzed Call Reasons (Billing, Payments, Service Outage).
- **Channel Distribution:** Break down of call volume by channel (Call-Center, Chatbot, Email, Web).
- **Geographical Analysis:** Sales and call volume distribution across different US states.

## Technical Skills Applied
- **Advanced Excel:** Used Pivot Tables, Slicers, and advanced charting for interactivity.
- **Data Visualization:** Designed a professional dark-themed dashboard for better readability.
- **Data Cleaning:** Processed raw call logs into a structured format for analysis.
